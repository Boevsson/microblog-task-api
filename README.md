# microblog-task-vue

## Project setup
```
1. composer install

2. You can see the database configuration in cofig/settings.php and change it. 

3. Use the TestSeedData.sql in the tests folder to set up the database

4. Run tests using php vendor/bin/phpunit

5. You can see all the avalable routes in src/routes.php

6. You can login with:
{
    "email": "test@email.com",
    "password": "rocknroll"
}
```
